

## how To Use

run the following commands :

```bash
git clone https://github.com/Asalek/minishell.git
cd minishell
./brew_installer.bash
brew install openssl@1.1
brew install readline
make && clear
./minishell
```

brew install openssl@1.1

brew install readline
